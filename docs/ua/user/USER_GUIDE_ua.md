## Початок роботи

### Перший вхід

1. Перейдіть на `https://award-system.edu.ua`
2. Натисніть **"Увійти з обліковим записом університету"**
3. Введіть вашу університетську електронну пошту та пароль
4. Пройдіть двофакторну автентифікацію (якщо увімкнено)
5. Прийміть Умови використання та Політику конфіденційності

### Огляд панелі керування

Після входу ви побачите особисту панель керування:

| Розділ | Опис |
|--------|------|
| **Мої нагороди** | Нагороди, які ви подали |
| **Очікують дій** | Елементи, що потребують вашої уваги |
| **Остання активність** | Останні оновлення ваших нагород |
| **Швидка статистика** | Підсумок вашої статистики нагород |

## Подання нагород

### Крок 1: Почати нове подання

1. Натисніть кнопку **"+ Нова нагорода"**
2. Виберіть категорію нагороди:
   - Міжнародна (конференції, публікації)
   - Національна (конкурси, гранти)
   - Регіональна (місцеві досягнення)
   - Локальна (університетський рівень)

### Крок 2: Заповнити деталі нагороди

| Поле | Опис | Обов'язкове |
|------|------|-------------|
| Назва | Назва нагороди точно як на сертифікаті | Так |
| Опис | Короткий опис досягнення | Ні |
| Дата нагородження | Дата отримання (РРРР-ММ-ДД) | Так |
| Організація-нагороджувач | Хто надав нагороду | Так |
| Категорія | Вибрати з випадаючого списку | Так |

### Крок 3: Завантажити підтверджуючі документи

**Обов'язкові документи:**
- Сертифікат або диплом (PDF, якісний скан)

**Додаткові документи:**
- Програма заходу
- Витяг з публікації
- Фотографії
- Додаткові підтверджуючі матеріали

**Вимоги до файлів:**
- Формати: PDF, JPEG, PNG
- Максимальний розмір: 10 МБ на файл
- Рекомендовано: PDF з роздільною здатністю 300 DPI

### Крок 4: Перевірити та подати

1. Перевірте всю введену інформацію
2. Переконайтеся, що завантажені документи читабельні
3. Натисніть **"Подати на розгляд"**
4. Підтвердіть подання у спливаючому вікні

### Після подання

- Статус нагороди змінюється на **"На розгляді"**
- Ви отримаєте підтвердження електронною поштою
- Відстежуйте прогрес у розділі **"Мої нагороди"**

## Керування документами

### Завантаження документів

```
Підтримувані формати:
├── PDF (рекомендовано для сертифікатів)
├── JPEG (для фотографій)
└── PNG (для скріншотів)
```

**Поради для найкращих результатів:**
- Використовуйте скани високої роздільної здатності (300 DPI)
- Переконайтеся, що текст читабельний
- Тримайте розмір файлів до 5 МБ для швидшого завантаження
- Використовуйте описові назви файлів

### Перегляд документів

1. Перейдіть до деталей нагороди
2. Натисніть на мініатюру документа
3. Використовуйте елементи керування для збільшення/повороту
4. Завантажте оригінал за потреби

### Заміна документів

1. Відкрийте нагороду зі статусом **Чернетка**
2. Натисніть **"×"** біля документа для видалення
3. Завантажте новий документ
4. Збережіть зміни

## Відстеження статусу нагород

### Потік статусів

```
Чернетка → Подано → На розгляді → Схвалено/Відхилено → Опубліковано
                         ↓
                  Повернуто на доопрацювання
```

### Значення статусів

| Статус | Опис | Ваші дії |
|--------|------|----------|
| **Чернетка** | Ще не подано | Завершити та подати |
| **Подано** | Очікує розгляду | Чекати на рецензента |
| **На розгляді** | Оцінюється | Чекати на рішення |
| **Повернуто** | Потребує доопрацювання | Внести запитані зміни |
| **Схвалено** | Прийнято | Нічого - буде опубліковано |
| **Відхилено** | Не прийнято | Переглянути причину |
| **Опубліковано** | Видиме публічно | Нічого - завершено |

### Перевірка статусу

1. Перейдіть до **"Мої нагороди"**
2. Перегляньте колонку статусу або фільтруйте за статусом
3. Натисніть на нагороду для детальної історії статусу

## Керування профілем

### Оновлення персональної інформації

1. Натисніть ваше ім'я у верхньому правому куті
2. Виберіть **"Налаштування профілю"**
3. Оновіть доступні поля:
   - Відображуване ім'я
   - Контактна інформація
   - Налаштування сповіщень

### Налаштування конфіденційності

| Налаштування | Опції |
|--------------|-------|
| Видимість email | Публічний / Тільки організація / Приватний |
| Відображення нагород | Показати всі / Тільки схвалені |
| Видимість профілю | Публічний / Тільки організація |

## Сповіщення

### Типи сповіщень

| Тип | Опис | За замовчуванням |
|-----|------|-----------------|
| Зміна статусу | Нагорода схвалена/відхилена/повернута | Email + Браузер |
| Запит на розгляд | Нагорода потребує вашого розгляду | Email + Браузер |
| Нагадування | Нагадування про очікувані елементи | Щотижневий email |
| Системне | Обслуговування, оновлення | Тільки email |

### Керування сповіщеннями

1. Перейдіть до **Налаштування профілю** → **Сповіщення**
2. Увімкніть/вимкніть сповіщення за типом
3. Виберіть метод доставки (email/браузер/обидва)
4. Встановіть години тиші за потреби

## Часті запитання

### Загальні

**П: Хто може бачити мої подані нагороди?**  
В: Нагороди зі статусом "Опубліковано" видимі публічно. Чернетки та очікувані нагороди видимі лише вам та рецензентам.

**П: Чи можу я редагувати нагороду після подання?**  
В: Ні, подані нагороди не можна редагувати. Якщо потрібні зміни, рецензент поверне її на доопрацювання.

**П: Скільки триває процес розгляду?**  
В: Зазвичай 3-5 робочих днів для кожного рівня розгляду (Кафедра → Факультет → Університет).

### Проблеми з поданням

**П: Мій файл не завантажується. Що робити?**  
В: Перевірте:
1. Файл у форматі PDF, JPEG або PNG
2. Розмір файлу до 10 МБ
3. Назва файлу не містить спеціальних символів
4. Браузер оновлений

**П: Я подав неправильний документ. Як виправити?**  
В: Зверніться до секретаря вашої кафедри або системного адміністратора для повернення нагороди на доопрацювання.

### Проблеми з обліковим записом

**П: Я забув пароль. Що робити?**  
В: Натисніть "Забули пароль" на сторінці входу та дотримуйтесь інструкцій в email.

**П: Мій обліковий запис заблоковано. Як розблокувати?**  
В: Після 5 невдалих спроб входу облікові записи блокуються на 30 хвилин. Зверніться до IT підтримки для негайного доступу.

### Дані та конфіденційність

**П: Як захищені мої дані?**  
В: Всі дані зашифровані, зберігаються безпечно та відповідають GDPR. Див. Політику конфіденційності.

**П: Чи можу я видалити свої дані?**  
В: Так, ви можете запросити видалення даних згідно з GDPR. Перейдіть до Профіль → Конфіденційність → "Запросити видалення даних".

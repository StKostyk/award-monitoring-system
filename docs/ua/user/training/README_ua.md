## Посібники швидкого старту

### Для кінцевих користувачів

```
Швидкий старт: Подання першої нагороди (5 хвилин)

1. Вхід → Натисніть "Нова нагорода"
2. Заповніть: Назва, Категорія, Дата, Організація
3. Завантажте сертифікат (PDF/JPEG)
4. Натисніть "Подати на розгляд"
5. Готово! Відстежуйте статус у "Мої нагороди"
```

### Для рецензентів

```
Швидкий старт: Обробка черги перегляду (5 хвилин)

1. Вхід → Перейдіть до "Черга перегляду"
2. Натисніть на очікувану нагороду
3. Перегляд: Деталі, Документи, Історія
4. Рішення: Схвалити / Відхилити / Повернути
5. Додайте коментар → Підтвердіть рішення
```

### Для адміністраторів

```
Швидкий старт: Додавання нового користувача (3 хвилини)

1. Адміністрування → Користувачі → Додати користувача
2. Введіть: Email, Ім'я, Роль, Організація
3. Натисніть "Створити користувача"
4. Користувач отримає email активації
```

## Практичні вправи

### Вправа 1: Подання тестової нагороди

**Мета**: Практика робочого процесу подання нагороди

**Кроки**:
1. Увійдіть у навчальне середовище
2. Створіть нову нагороду з тестовими даними
3. Завантажте зразок сертифіката
4. Подайте на розгляд
5. Перевірте email підтвердження

**Критерії успіху**: Нагорода з'являється в "Мої нагороди" зі статусом "Подано"

### Вправа 2: Перегляд та затвердження

**Мета**: Практика робочого процесу перегляду

**Передумови**: Завершити Вправу 1

**Кроки**:
1. Увійдіть як рецензент (навчальний обліковий запис)
2. Знайдіть подану нагороду в черзі
3. Перегляньте документацію
4. Схваліть з коментарем
5. Перевірте зміну статусу

**Критерії успіху**: Статус нагороди змінюється на "Схвалено"

### Вправа 3: Генерація звіту

**Мета**: Практика функцій звітування (Адміністратори)

**Кроки**:
1. Увійдіть як адміністратор
2. Перейдіть до розділу Звіти
3. Виберіть звіт "Підсумок нагород"
4. Встановіть діапазон дат (останні 30 днів)
5. Експортуйте як PDF

**Критерії успіху**: PDF звіт успішно завантажується

## Оцінювання

### Сертифікація кінцевого користувача

**Вимоги**:
- Завершити всі модулі Напрямку 1

**Вимоги**:
- Завершити всі модулі Напрямку 2

- Правильно обробити 3 практичних перегляди

- Завершити всі модулі Напрямку 3

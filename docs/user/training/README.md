## Quick Start Guides

### For End Users

```
Quick Start: Submit Your First Award (5 minutes)

1. Login → Click "New Award"
2. Fill in: Title, Category, Date, Organization
3. Upload certificate (PDF/JPEG)
4. Click "Submit for Review"
5. Done! Track status in "My Awards"
```

### For Reviewers

```
Quick Start: Process Review Queue (5 minutes)

1. Login → Go to "Review Queue"
2. Click on pending award
3. Review: Details, Documents, History
4. Decision: Approve / Reject / Return
5. Add comment → Submit decision
```

### For Administrators

```
Quick Start: Add New User (3 minutes)

1. Administration → Users → Add User
2. Enter: Email, Name, Role, Organization
3. Click "Create User"
4. User receives activation email
```

## Hands-On Exercises

### Exercise 1: Submit Test Award

**Objective**: Practice award submission workflow

**Steps**:
1. Login to training environment
2. Create new award with test data
3. Upload sample certificate
4. Submit for review
5. Verify confirmation email

**Success Criteria**: Award appears in "My Awards" with status "Submitted"

### Exercise 2: Review and Approve

**Objective**: Practice review workflow

**Prerequisites**: Complete Exercise 1

**Steps**:
1. Login as reviewer (training account)
2. Find submitted award in queue
3. Review documentation
4. Approve with comment
5. Verify status change

**Success Criteria**: Award status changes to "Approved"

### Exercise 3: Generate Report

**Objective**: Practice reporting features (Admins)

**Steps**:
1. Login as administrator
2. Navigate to Reports section
3. Select "Awards Summary" report
4. Set date range (last 30 days)
5. Export as PDF

**Success Criteria**: PDF report downloads successfully
